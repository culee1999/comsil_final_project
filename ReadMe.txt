<MineSweeper>

<실행 환경>
Window 11, Visual Studio 2017

<키보드, 마우스>
'S' or 's' : 지뢰찾기 배경을 정사각형으로 설정
'H' or 'h' : 지뢰찾기 배경을 하트로 설정
'A' or 'a' : 정답을 cmd 창에 출력
'R' or 'r' : 리셋하고 재시작
마우스 왼쪽 클릭 : 블록 열기
마우스 오른쪽 클릭 : 깃발 그리기 또는 깃발 지우기

<게임 설명>
마우스 왼쪽, 오른쪽 클릭을 이용해 모든 지뢰를 찾는 것이 목표

<실행 방법>
게임 배경을 정사각형으로 두고 플레이하려면 s를 누르고, 하트로 두고 플레이하려면 h를 누른다.
'마우스 왼쪽 클릭으로 블록 오픈, 마우스 오른쪽 클릭으로 깃발 세우고 지우기' 과정을 반복한다.
지뢰를 3번 클릭하면 gameover.
숨겨진 블록과 깃발 개수가, 전체 지뢰 개수와 일치하면 game clear.
